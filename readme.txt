---------------------
ไฟล์แต่ละไฟล์ใช้ทำอะไรบ้าง
---------------------

Data (Folder) : เก็บข้อมูลไฟล์ csv ที่ใช้ในการคำนวณภายใน jupyter notebook

GeoVisualization (Folder) : ไฟล์ csv ที่ใช้ในการคำนวณใน part 4 ในส่วนของ Geo-visualization
6210503781_GeoVisualization(Google sheet) : นำข้อมูลที่ได้จาก part4.ipynb มาทำการสร้าง Geo-Visualization

part1.ipynb : 
ใช้ในการ clean ข้อมูลใน part1 (Boxplot) รวมถึงการคำนวณและการ plot กราฟด้วยโดยนำข้อมูลมาจาก Data(Folder)

part2.1.ipynb : 
ใช้ในการ clean ข้อมูลใน part2.1 (Histogram) รวมถึงการคำนวณและการ plot กราฟด้วยโดยนำข้อมูลมาจาก Data(Folder)
 
part2.2.ipynb : 
ใช้ในการ clean ข้อมูลใน part2.2(หาตารางคำนวณค่าต่างๆของ Thailand เทียบกับอีก 3 ทวีปที่ได้รับมา) รวมถึงการคำนวณด้วยและการ plot กราฟด้วยโดยนำข้อมูลมาจาก Data(Folder)
 
part3.ipynb : 
ใช้ในการ clean ข้อมูลใน part3 (correlation) รวมถึงการคำนวณด้วยและการ plot correlation ด้วยโดยนำข้อมูลมาจาก Data(Folder)

part4.ipynb : 
ใช้ในการ clean ข้อมูลใน part4 รวมถึงการคำนวณด้วยและการ plot กราฟ Time-series และ Deviation นอกจากนั้นยัง export ไฟล์บางไฟล์ที่ต้องนำไปใช้ในการสร้าง Geo-visualization ในไฟล์ GeoVisualization
